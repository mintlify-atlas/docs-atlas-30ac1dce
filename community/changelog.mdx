## Version 2025.001.000 (2025-10-25)

<Warning>
  **Major Rewrite**: This release represents a complete rewrite of sunnypilot with significant changes to core systems.
</Warning>

### Major Systems Rewrite

- Fully adopts upstream comma.ai openpilot, opendbc, and panda test suites
- Added regression testing for stability
- Aligns with comma.ai's safety policy
- Some features temporarily disabled pending reimplementation

### Major Features

<AccordionGroup>
  <Accordion title="Modular Assistive Driving System (MADS)">
    Complete driving assistance framework with enhanced engagement modes
  </Accordion>
  <Accordion title="Driving Model Manager">
    Custom driving model selection supporting ~86 models from Night Strike (Oct 2023) to The Cool People's Models (Oct 2025)
  </Accordion>
  <Accordion title="Neural Network Lateral Control (NNLC)">
    Advanced torque-based lateral control (formerly NNFF)
  </Accordion>
  <Accordion title="Dynamic Experimental Control (DEC)">
    Intelligent longitudinal control adaptation between ACC and E2E
  </Accordion>
  <Accordion title="Speed Limit Assist (SLA)">
    Comprehensive speed limit integration featuring:
    - @pfeiferj's mapd for offline map limits
    - Speed Limit Resolver (car, map, combined sources)
    - On-screen UI for Speed Limit Information/Warning
    - Automatic cruise speed adjustment
  </Accordion>
  <Accordion title="Intelligent Cruise Button Management (ICBM)">
    Manages vehicle speed via cruise control button commands to car ECU
  </Accordion>
  <Accordion title="Smart Cruise Control (SCC-M / SCC-V)">
    Controls turn entry speed using:
    - SCC-M: Map data
    - SCC-V: Vision model curve detection
  </Accordion>
  <Accordion title="Vehicle Selector">
    Manual vehicle selection when automatic fingerprinting fails
  </Accordion>
  <Accordion title="sunnylink Integration">
    Cloud connectivity with settings backup/restore. Remote management infrastructure ready for future releases.
  </Accordion>
  <Accordion title="mapd Integration">
    OpenStreetMap database downloads for Speed Limit Assist (thanks to @pfeiferj)
  </Accordion>
</AccordionGroup>

### User Interface

- **Complete UI Redesign** from default openpilot experience
- **New Settings Panels**: Steering, Longitudinal, Vehicle, Models, Visuals, Display, Trips
- **Advanced Controls Toggle**: Simplified default settings with option for advanced features
- **Models Panel**: Download manager, folders, favorites, fuzzy search, cache refresh
- **Branch & Platform Selectors**: Searchable branch selector and platform display
- **Developer UI**: Better alert positioning and integrated error log viewer
- **OpenStreetMap Downloader**: Search feature for finding areas

### Visuals & Display

- Brightness controls
- Custom interactivity timeouts
- Green light indicator
- Lead vehicle indicator
- On-screen turn signals
- Blind spot indicators
- Lead chevron info
- Standstill timer
- Road name display
- Rainbow road path (Tesla-like)
- Screen off while driving with customizable wake behavior

### Control Systems

- **Pause Lateral on Blinker**: Temporary pause when turn signal active
- **Custom ACC Setpoint Increments**: Configurable speed adjustment increments
- **Steering on Brake Press**: Customizable steering behavior
- **Enforce Torque Lateral Control**: Fine-tuning settings
- **Automatic Lane Change**: Support with disable option

### Technical Infrastructure

- **Custom Cereal**: Dedicated cereal for sunnypilot-specific events
- **Car Interface Abstractions**: Brand-specific settings support
- **Param Store Caching**: Reduced startup times
- **Enhanced Error Handling**: Improved Sentry logging
- **Docker & CI/CD**: Full automation for testing, building, and releasing

- Display DM Camera in Reverse Gear
